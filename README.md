# consult

Cross-tool AI consultation: get second opinions from Gemini, Codex, Claude, OpenCode, or Copilot CLI

## Installation

```bash
# Claude Code
claude mcp add-plugin agent-sh/consult

# Or install from marketplace
agentsys install consult
```

## Usage

```
/consult
```
